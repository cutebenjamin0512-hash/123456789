# 📚 背單字軟體

一個純前端的背單字應用，使用 HTML、CSS 和 JavaScript 開發。無需框架，簡潔輕量！

## ✨ 功能特點

### 1. **學習頁面**
- 🎴 中央卡片設計，點擊可翻轉
- **正面**：顯示英文單字
- **背面**：顯示翻譯、詞性、例句、字根分析
- ⬅️ ➡️ 上一個/下一個按鈕快速導航
- ⌨️ 快捷鍵支持：
  - `←` / `→` 鍵：上一個/下一個單字
  - `空格鍵`：翻轉卡片

### 2. **管理頁面**
- ➕ 新增單字表單
- 🔍 **自動填入按鈕**：調用 API 自動填入單字信息
  - 翻譯
  - 詞性
  - 例句
  - 字根分析
- 📋 已新增單字列表
- 🗑️ 刪除單字功能

### 3. **數據存儲**
- 💾 使用 LocalStorage 存儲本地數據
- 📱 關閉瀏覽器後數據不會丟失
- 🌐 純前端，無需後端服務器

## 🚀 快速開始

### 在本地運行

1. **下載或克隆項目**
   ```bash
   git clone <repository-url>
   cd word-card-app
   ```

2. **打開應用**
   - 直接雙擊 `index.html` 打開
   - 或者使用 HTTP 服務器：
     ```bash
     # Python 3
     python -m http.server 8000
     
     # Python 2
     python -m SimpleHTTPServer 8000
     
     # Node.js (使用 http-server)
     npm install -g http-server
     http-server
     ```

3. **在瀏覽器中訪問**
   - 直接打開本地文件或訪問 `http://localhost:8000`

## 📖 使用指南

### 新增單字

#### 方式 1：手動填入
1. 點擊「管理單字」tab
2. 在表單中填入所有欄位
3. 點擊「✨ 新增單字」按鈕

#### 方式 2：自動填入（推薦）
1. 點擊「管理單字」tab
2. 在「英文單字」欄輸入單字
3. 點擊「🔍 自動填入」按鈕
4. 應用會自動調用 API 填入其他信息
5. 檢查無誤後點擊「✨ 新增單字」

### 學習模式
1. 點擊「學習」tab
2. 點擊卡片翻轉查看背面
3. 使用按鈕或鍵盤快捷鍵導航
4. 下方顯示進度：「當前 / 總數」

### 管理單字
1. 在「管理單字」頁面可查看所有已新增單字
2. 點擊「🗑️ 刪除」按鈕刪除不需要的單字

## 🔧 技術棧

- **HTML5**：頁面結構
- **CSS3**：樣式和3D翻轉效果
  - CSS Transform 實現 3D 翻頁
  - Flexbox 布局
  - 漸變背景
- **JavaScript**：應用邏輯
  - Fetch API 調用外部 API
  - LocalStorage 數據持久化
  - DOM 事件監聽

## 📚 API 來源

使用免費的 **Dictionary API**：
- **API 地址**：https://api.dictionaryapi.dev/api/v1/entries/english
- **用途**：自動獲取單字的定義、詞性、例句等信息
- **特點**：無需認證密鑰，完全免費
